### Overview:
The data analysis presented in this report aims to answer the following three questions regarding CO<sub>2</sub> emissions:<br><br>

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;***Q1.&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;What is the biggest predictor of a large CO<sub>2</sub> output per capita of a country?<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Q2.&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Which countries are making the biggest strides in decreasing CO<sub>2</sub> output?<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Q3.&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Which non-fossil fuel energy technology will have the best price in the future?***<br><br>

**Jupyter Notebooks**:<br>
All data analyses to answer these questions were done with custom Python code in two Jupyter Notebooks.  These Notebooks are supplements to this report and can also be found in this Github repository:
- **Supplement A**: Python code & data analysis to answer **Q1 and Q2**
- **Supplement B**: Python code & data analysis to answer **Q3**<br><br>

**Datasets used**:<br>
- The analysis exclusively used datasets from the **<a href= https://ourworldindata.org/ target="_blank">"Our World in Data" website</a>**.
- All downloaded datasets can be found in this Github repository in the folder "Datasets".<br><br>

### Main findings:<br>
- You can find a **3-page Executive Summary** in the <a href="https://github.com/Dolphinity-io/CO2-emissions/raw/main/CO2_Report_Dolphinity__June_2023.pdf" target="_blank">PDF Download of the Report</a>, with answers to each of the three questions, as well as a description of the general approach and assumptions made.<br><br>
- The charts and table below provide a useful overview of the main findings, which are discussed with additional context in the Report.  Many additional charts can be found in Appendix 2 of the <a href="https://github.com/Dolphinity-io/CO2-emissions/raw/main/CO2_Report_Dolphinity__June_2023.pdf" target="_blank">Report</a> as well.<br><br>
- In addition, **Appendix 1** contains a literature review, discussing potential causes as to why the extrapolated **price forecast trend for Solar PV seems to have broken** in the past two years (2021-2023).<br><br>
